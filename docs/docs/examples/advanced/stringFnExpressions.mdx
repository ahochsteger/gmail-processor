This example shows how to use string functions within expressions.

The following string functions are supported:
- replace(searchRegexp, replacement)
- replaceAll(searchRegexp, replacement)
- slice(start, end)
- substring(start, end)
- trimStart()
- trimEnd()
- trim()
- toUpperCase()
- toLowerCase()
